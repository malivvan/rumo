# ISSUES.md ‚Äî Combined Architectural Issue Registry

Unified view of all architectural/systemic issues identified across the `rumo` project.
Each issue has a unique sequential ID. Sorted descending by severity (critical ‚Üí high ‚Üí medium ‚Üí low).

**Sources:**
- `FIXME.md` ‚Äî root package + `cmd` CLI
- `vm/FIXME.md` ‚Äî VM, compiler, object system, concurrency
- `std/FIXME.md` ‚Äî stdlib module framework + cross-module concerns
- `std/cli/FIXME.md` ‚Äî CLI module (cobra-based)
- `std/cui/FIXME.md` ‚Äî TUI module (tview-based)
- `std/shell/FIXME.md` ‚Äî readline/shell module

**Severity:** critical (crashes, data races, build failures under normal use) ¬∑ high (subtle bugs, silent misbehaviour) ¬∑ medium (performance, maintainability, usability) ¬∑ low (minor improvements)

**Related issues** are noted with `‚Üí #N` cross-references.
